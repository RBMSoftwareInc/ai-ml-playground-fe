## üîå API Endpoints

### Use Case Execution

**Endpoint**: `POST /api/use-cases/:id/execute`

**Request Body**:
```json
{
  "scenario": "stolen-card",
  "payload": {
    "amount": 89999,
    "location": "Mumbai",
    "previous_location": "Delhi"
  },
  "mode": "business" | "tech"
}
```

**Response**:
```json
{
  "success": true,
  "result": {
    "status": "fraud_detected",
    "confidence": 0.92
  },
  "pipeline": [
    {
      "id": "feature_extract",
      "label": "Feature Extraction",
      "latency_ms": 45,
      "confidence": 0.95
    }
  ],
  "metrics": {
    "latency_ms": 245,
    "confidence": 0.92,
    "tokens_consumed": 1250
  },
  "trace": {}
}
```

### Solution Configuration

**Location**: `/data/solutions/[solution-id].json`

**Structure**:
- Solution metadata (id, title, tagline, icon)
- Impact metrics
- Business story
- Business/Tech mode configurations
- Scenarios with dynamic form fields
- Processing pipeline steps
- Metrics display settings
